## ðŸ“± How to Use the App

### Starting an Exam
1. Click **"Start Mock Exam"** on home page
2. Review exam information
3. Click **"Start Exam"** button
4. Exam begins with 60-minute timer

### During the Exam

#### **Reading Questions**:
1. Category shown at top
2. Question text in center
3. Four options (A, B, C, D) below

#### **Answering**:
1. Click an option to select
2. Selected option: Black background
3. Can change answer anytime

#### **Flagging**:
1. Click **"Flag Question"** button
2. Button changes to **"Flagged for Review"**
3. Small flag appears on question number
4. Click again to unflag

#### **Navigating**:
- **Sidebar**: Click any question number (1-40)
- **Bottom**: Use Previous/Next buttons
- **Filters**: Show All/Flagged/Done/Todo
- **Quick Nav**: Jump to first unanswered/flagged

#### **Using Tabs**:
- **Tips**: Read exam strategies
- **Rules**: Check exam rules
- **Review**: See unanswered/flagged lists

### Before Submitting
1. Click **"Review"** tab
2. Check **Unanswered** section
3. Check **Flagged** section
4. Click numbers or "Go" to jump
5. Answer remaining questions
6. Click **"Submit Exam"** on last question

### After Submission
1. See your score (percentage)
2. View correct/incorrect count
3. Pass/Fail status (need 65%)
4. Review all questions with answers
5. Retake or go to practice mode
